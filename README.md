# DriversInterfaz

DriversInterfaz es una aplicación JavaFX que muestra los resultados de los conductores a lo largo de diferentes años en un formato tabular. Los usuarios pueden seleccionar un año de un menú desplegable, y la aplicación mostrará los resultados correspondientes de los conductores, incluyendo su nombre, número de victorias, puntos totales y clasificación.

## Características

- Selección de año desde un ComboBox.
- Visualización de resultados de conductores para el año seleccionado en un TableView.
- La tabla muestra el nombre del conductor, las victorias, los puntos totales y la clasificación.

## Requisitos

- Java 8 o superior
- Biblioteca JavaFX
